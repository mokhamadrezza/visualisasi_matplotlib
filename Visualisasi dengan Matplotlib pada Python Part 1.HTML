Pengenalan Matplotlib

Matplotlib adalah salah satu pustaka (library) Python yang paling populer untuk visualisasi data. Dikenal karena kemampuannya untuk menghasilkan grafik yang berkualitas tinggi dan mendukung berbagai jenis plot, Matplotlib menjadi alat penting bagi data scientist, peneliti, dan siapa saja yang bekerja dengan data.

Matplotlib memungkinkan pembuatan berbagai plot dengan mudah, termasuk plot garis, histogram, batang, kotak, dan radial, dengan fleksibilitas tinggi untuk gaya yang disempurnakan dan anotasi yang disesuaikan.

Tampilan Matplotlib

Pada proses awal pembuatan visualisasi, pengguna perlu mendefinisikan canvas terlebih dahulu, dengan cara meng-import library matplotlib. Ini akan memungkinkan pengguna untuk menggunakan berbagai fungsi dan metode yang tersedia dalam matplotlib untuk membuat dan menyesuaikan grafik.

Berikut adalah contoh kode python untuk matplotlib:

# Import library matplotlib
import matplotlib.pyplot as plt
fig, ax = plt.subplots()
plt.show()

plt.subplots() memiliki fungsi untuk mendefinisikan canvas dan letaknya di mana, fig adalah canvas sedangkan, ax adalah letak posisi gambar dalam canvas.

Pada matplotlib garis horizontal merupakan sumbu X sedangkan garis vertikal adalah sumbu Y.

Axes pada Matplotlib

Dalam konteks Matplotlib, "axes" mengacu pada area di dalam figure di mana grafik digambar. Setiap figure dapat memiliki satu atau lebih axes, dan masing-masing axes dapat berisi satu grafik atau lebih. Axes adalah unit paling dasar dan fleksibel untuk membuat subplot. Axes memungkinkan penempatan plot di lokasi mana pun.

Berikut adalah contoh kode penggunaan axes pada matplotlib:

import matplotlib.pyplot as plt

# Membuat figure dan axes
fig, ax = plt.subplots(figsize=(8, 6)) # Ukuran canvas

# Data
x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]

# Membuat plot pada axes
ax.plot(x, y, marker='o')

# Menambahkan judul dan label
ax.set_title('Contoh Axes dengan subplots')
ax.set_xlabel('X Axis')
ax.set_ylabel('Y Axis')

# Menampilkan grafik
plt.show()

Untuk menambahkan banyak plot dalam satu gambar pengguna hanya perlu memanggil fungsi ax.plot() dalam 1 cell code yang sama sebanyak plot yang ingin ditampilkan dalam satu gambar selanjutnya tinggal menyesuaikan data yang ingin ditampilkan dalam satu gambar.

Syntax di bawah ini adalah contoh membuat banyak plot dalam satu gambar.

import matplotlib.pyplot as plt

# Membuat figure dan axes
fig, ax = plt.subplots(figsize=(8, 6)) # Ukuran canvas

# Data
x = [1, 2, 3, 4, 5]
y1 = [2, 3, 5, 7, 11] # Data pertama
y2 = [1, 4, 6, 8, 10] # Data kedua

# Membuat plot pada axes
ax.plot(x, y1, marker='o', label='Data 1') # Plot pertama
ax.plot(x, y2, marker='s', label='Data 2') # Plot kedua

# Menambahkan judul dan label
ax.set_title('Contoh Axes dengan Banyak Plot')
ax.set_xlabel('X Axis')
ax.set_ylabel('Y Axis')
ax.legend() # Menampilkan legenda

# Menampilkan grafik
plt.show()

Output yang dihasilkan adalah sebagai berikut:

Cara yang sama juga dapat dimanfaatkan untuk membuat banyak plot dari satu data dengan berbagai kolom yang ingin divisualisasikan dalam satu gambar, dengan cara memanggil ax.plot berkali kali dalam satu cell yang sama kemudian tinggal menyesuaikan kolom apa yang akan ditampilkan dalam satu plot.

# Membuat figure dan axes

ax.plot(x, y3, marker='^', label='Data 3') # Plot ketiga

# Menambahkan judul dan label
ax.set_title('Contoh Axes dengan Banyak Plot')
ax.set_xlabel('X Axis')
ax.set_ylabel('Y Axis')

# Menambahkan legenda

Mengatur Plot Matplotlib

Dalam Matplotlib, plot dapat diatur sesuai dengan kebutuhan pengguna melalui berbagai parameter dan fungsi yang tersedia. Mulai dari pengaturan tampilan seperti warna, garis, marker, hingga pengaturan label pada sumbu-sumbu X dan Y, serta judul plot.

Pengguna dapat melakukan penyesuaian gambar plot dengan menambahkan kata kunci perintah dalam matplotlib, seperti menambahkan marker dengan warna dan bentuk yang bisa disesuaikan dan membuat garis putus-putus pada plot garis.

Berikut adalah contoh kode mengatur plot matplotlib:

import matplotlib.pyplot as plt

# Membuat figure dan axes
fig, ax = plt.subplots(figsize=(8, 6)) # Ukuran canvas

# Data
x = [1, 2, 3, 4, 5]
y1 = [2, 3, 5, 7, 11] # Data pertama
y2 = [1, 4, 6, 8, 10] # Data kedua

# Membuat plot pada axes dengan warna
ax.plot(x, y1, marker='o', label='Data 1', color='blue') # Plot pertama
ax.plot(x, y2, marker='s', label='Data 2', color='orange') # Plot kedua

# Menampilkan grafik
plt.show()

Subplot pada Matplotlib

Terkadang dalam membuat banyak plot dalam satu kanvas tidaklah efektif karena plot akan saling tumpang tindih. Untuk mengatasi ini, pada matplotlib telah menyediakan sebuah method untuk memisahkan gambar pada kanvas berbeda, method ini disebut dengan subplots.

Subplot berarti group axes yang terdapat dalam satu gambar matplotlib. subplots() berfungsi membantu dalam membuat beberapa tata letak dan memberikan kontrol atas semua plot individu yang dibuat.

Dalam membuat banyak gambar dalam satu kanvas, dibutuhkan fungsi plt.subplots(nrow,ncol) dari matplotlib di mana nrow menunjukkan jumlah baris yang ingin dibuat, dan ncol menunjukkan jumlah kolom yang ingin dibuat dalam satu canvas. Berikut adalah contoh membuat kanvas yang memuat 6 gambar dengan rincian 3 baris dan 2 kolom:

#membuat subplots 3 baris dan 2 kolom
fig,ax = plt.subplots(3,2)

#memanggil plot
plt.show()

Output yang dihasilkan adalah sebagai berikut:

Setelah mengatur banyak gambar yang akan dimuat dalam satu canvas, maka tahap selanjutnya adalah memasukkan plot pada tiap-tiap gambar dalam canvas dengan mengatur penulisan index untuk mengatur tata letak plot dalam sebuah gambar. Berikut adalah contoh penulisan ax[0,0] yang memiliki makna bahwa plot yang akan dibuat dimasukkan ke dalam gambar pada letak baris ke 0 dan kolom ke 0.

import matplotlib.pyplot as plt
import numpy as np

# Data contoh
x = np.linspace(0, 10, 100)
y = np.sin(x)

# Membuat subplots 3 baris dan 2 kolom
fig, ax = plt.subplots(3, 2)

# Menambahkan plot pada posisi (0, 0) saja
ax[0, 0].plot(x, y, color='blue')

# Menampilkan plot
plt.show()

Output yang dihasilkan adalah sebagai berikut:

Membuat banyak plot dalam satu gambar dari berbagai data juga mungkin dilakukan dalam subplot. Berikut adalah contoh membuat subplot terdiri dari 2 baris dan 1 kolom, dan membuat plot dari dua data berbeda:

import matplotlib.pyplot as plt
import numpy as np

# Data contoh
x = np.linspace(0, 10, 100)
y1 = np.sin(x)
y2 = np.cos(x)
y3 = np.tan(x) * 0.1

# Membuat subplots 2 baris dan 1 kolom
fig, ax = plt.subplots(2, 1)

# Subplot pertama: beberapa garis
ax[0].plot(x, y1, color='blue', label='Sin(x)')
ax[0].plot(x, y2, color='red', label='Cos(x)')
ax[0].plot(x, y3, color='green', label='Tan(x)')

# Subplot kedua: beberapa garis
ax[1].plot(x, y1**2, color='purple', label='Sin^2(x)')
ax[1].plot(x, y2**2, color='orange', label='Cos^2(x)')
ax[1].plot(x, y3**2, color='brown', label='Tan^2(x)')

# Menampilkan plot
plt.show()

Output yang dihasilkan adalah sebagai berikut:

Pada penulisan ax[0] dan ax[1] dituliskan sebanyak tiga kali, hal ini mengindikasikan telah meletakkan tiga plot dalam satu gambar.

Visualisasi Data Time Series dengan Matplotlib

Time Series adalah serial dari kumpulan data yang teratur oleh urutan waktu. Frekuensi urutan waktu yang dimiliki oleh data time series bisa meliputi tahunan, bulanan, jam, atau bahkan mili-detik. Selama data tersebut disimpan dalam urutan waktu, data itu adalah data time series.

Pada contoh di bawah ini akan digunakan parameter parse_dates yang berfungsi mengubah tipe data yang berbasis waktu menjadi tipe timestamp dan index_col untuk membuat kolom menjadi sebuah index pada dataset.

#import data climate_change.csv
import pandas as pd

climate_change = pd.read_csv("climate_change.csv", parse_dates=["date"], index_col="date")

Pengaturan Tampilan pada Visualisasi Data Time Series

Pengaturan tampilan ini memungkinkan pengguna untuk menyesuaikan penampilan visual dari plot sesuai dengan preferensi atau kebutuhan analisis mulai dari warna plot dan warna ticks. Selain menampilkan dua sumbu Y berbeda dalam satu gambar, pengguna juga dapat mengatur warna garis pada line plot salah satu atau dua-duanya dalam satu gambar, dengan menambahkan parameter color dalam ax dan ax2. Berikut adalah contoh kode penggunaan parameter color dalam ax dan ax2 pada python:

# Membuat subplot
fig, ax = plt.subplots()

# Membuat plot
ax.plot(climate_change.index, climate_change["co2"], color='blue')
ax.set_xlabel('Time')
ax.set_ylabel('CO2 (ppm)', color='blue')

# Membuat dua sumbu-y
ax2 = ax.twinx()
ax2.plot(climate_change.index, climate_change["relative_temp"], color='red')
ax2.set_ylabel('Relative temperature (Celsius)', color='red')

# Memanggil plot
plt.show()

Output yang dihasilkan adalah sebagai berikut:

Selain mengatur warna tiap plot dalam satu gambar, pengguna juga dapat mengatur warna pada sumbu Y, dengan menggunakan fungsi tick_params() dengan menambahkan parameter colors untuk mengatur warna pada tiap sumbu Y. Berikut adalah contoh penggunaan tick_params() pada python:

# Membuat subplot
fig, ax = plt.subplots()

# Membuat plot
ax.plot(climate_change.index, climate_change["co2"], color='blue')
ax.set_xlabel('Time')
ax.set_ylabel('CO2 (ppm)', color='blue')
ax.tick_params('y', colors='blue')

ax2.set_ylabel('Relative temperature (Celsius)', color='red')
ax2.tick_params('y', colors='red')

# Memanggil plot
plt.show()

Fungsi untuk Plot Time Series

Membangun plot time series atau plot lainnya secara berulang dapat memakan waktu yang cukup lama, untuk itu pengguna dapat memanfaatkan fungsi pada python untuk untuk membuat plot time series secara berulang.

# Mendefiniskan fungsi dengan nama plot_timeseries
def plot_timeseries(axes, x, y, color, xlabel, ylabel):
axes.plot(x, y, color=color) # Membuat plot
axes.set_xlabel(xlabel) # Membuat label sumbu-x
axes.set_ylabel(ylabel, # Membuat label sumbu-y
color=color) # Memberikan warna pada plot
axes.tick_params('y',
colors=color) # Memberikan warna pada sumbu-y

Dengan adanya fungsi tersebut, pengguna cukup memanggilnya saja untuk membuat plot time series tanpa perlu menuliskan kembali sintaksnya seperti contoh berikut:

fig, ax = plt.subplots()
plot_timeseries(ax, climate_change.index, climate_change['co2'], 'blue','Time','CO2 (ppm)')
ax2 = ax.twinx()
plot_timeseries(ax2,
climate_change.index,
climate_change['relative_temp'],
'red','Time','Relative temperature (Celsius)')
plt.show()

Anotasi pada Visualisasi Data Time Series

Pada matplotlib pengguna dapat menampilkan text tertentu dalam sebuah plot dengan menggunakan fungsi .annotate(). Hal ini berfungsi untuk menampilkan suatu pesan tertentu pada event tertentu yang terjadi pada sebuah data.

Contoh di bawah ini adalah kode untuk menampilkan text >1 degree yang diletakkan pada koordinat 2015-10-06 pada sumbu x dan 1 pada sumbu-y sebagai sebuah pesan bahwa pada saat itu temperature bernilai di atas 1 derajat:

fig, ax = plt.subplots()
plot_timeseries(ax, climate_change.index, climate_change['co2'], 'blue','Time','CO2 (ppm)')
ax2 = ax.twinx()
plot_timeseries(ax2, climate_change.index,
climate_change['relative_temp'],
'red','Time',
'Relative temperature (Celsius)')
ax2.annotate(">1 degree", xy=[pd.Timestamp("2015-10-06"), 1])
plt.show()

Output yang dihasilkan adalah sebagai berikut:

fig, ax = plt.subplots()
plot_timeseries(ax, climate_change.index, climate_change['co2'], 'blue','Time','CO2 (ppm)')
ax2 = ax.twinx()
plot_timeseries(ax2, climate_change.index,

'red','Time',
'Relative temperature (Celsius)')
ax2.annotate(">1 degree",
xy=(pd.Timestamp('2015-10-06'), 1),
xytext=(pd.Timestamp('2008-10-06'), -0.2))

Output yang dihasilkan adalah sebagai berikut:

Pada penulisan text (pesan) tertentu pada plot, pengguna dapat menambahkan beberapa style, salah satunya adalah Arrow (tanda panah) untuk menunjukkan text (pesan) tersebut terjadi pada saat kapan. Penambahan Arrow (tanda panah) ini bisa dilakukan dengan menambahkan parameter arrowprops={} pada fungsi annotate.

fig, ax = plt.subplots()
plot_timeseries(ax, climate_change.index, climate_change['co2'], 'blue','Time','CO2 (ppm)')
ax2 = ax.twinx()
plot_timeseries(ax2, climate_change.index,
climate_change['relative_temp'],
'red','Time',
'Relative temperature (Celsius)')
ax2.annotate(">1 degree",
xy=(pd.Timestamp('2015-10-06'), 1),
xytext=(pd.Timestamp('2008-10-06'), -0.2), arrowprops={})
